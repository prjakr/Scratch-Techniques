Google Drive 設定

Google Cloud Consoleで取得したOAuth 2.0クライアントID

📋 Google Cloud Console 設定手順
  1. Google Cloud Console を開く
  2. 新しいプロジェクトを作成（または既存を選択）
  3. 「APIとサービス」→「ライブラリ」で Google Drive API を有効化
  4. 「APIとサービス」→「OAuth同意画面」を設定
    • ユーザータイプ: 外部
    • アプリ名: Scratch Techniques
    • スコープに drive.file を追加
    • テストユーザーに自分のGmailを追加
  5. 「認証情報」→「認証情報を作成」→「OAuthクライアントID」
    • アプリケーションの種類: ウェブアプリケーション
    • 承認済みのJavaScriptオリジンに http://localhost:8080 を追加
  6. クライアントIDをコピーして上に貼り付け

💡 ローカルで使う場合: python -m http.server 8080 でサーバーを起動し http://localhost:8080 でアクセス
